### HTTP 헤더 용도
- HTTP 전송에 필요한 모든 부가정보
- 메시지 바디의 내용, 메시지 바디의 크기, 압축, 인증, 요청 클라이언트, 서버 정보, 캐
  시 관리 정보
- 필요시 임의의 헤더 추가 가능
  - helloworld: hihi

### RFC723X
- Representation(표현) = representation Metadata + Representation Data
- 표현 = 표현 메타데이터 + 표현 데이터
