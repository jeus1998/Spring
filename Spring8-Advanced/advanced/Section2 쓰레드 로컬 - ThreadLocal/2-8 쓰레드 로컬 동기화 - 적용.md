# 쓰레드 로컬 동기화 - 적용

### LogTraceConfig 수정 

```java
@Configuration
public class LogTraceConfig {
    @Bean
    public LogTrace logTrace(){
        return new ThreadLocalLogTrace();
    }
}
```
